# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-03-27

### Added

- Initial plugin structure
- Shared conventions: error-handling, logging
- **`stitch` skill** — Google Stitch MCP integration for AI-powered UI design generation
  - Full API reference for all 11 MCP tools (project, screen, design system, variant operations)
  - Prompt guide with 3-Layer Framework (Anatomy + Vibe + Content)
  - 5 detailed workflows (full project, design exploration, iterative refinement, design-first, responsive pairs)
  - 6 ready-to-use design system templates (dark minimal, clean light, editorial, vibrant, monochrome, finance)
  - Gotchas for timing, ID types, and design system lifecycle
