# Support

## How to Get Help

### Documentation

- [README.md](README.md) — Quick start and overview
- [Plugin README](plugins/ctx/README.md) — Detailed CTX plugin documentation
- [CHANGELOG](plugins/ctx/CHANGELOG.md) — Version history and changes

### Ask a Question

- **GitHub Issues**: Open an issue with the "question" label at [GitHub Issues](https://github.com/tdp1999/claude-code-ctx/issues)
- **Email**: tdp99.business@gmail.com

### Report a Bug

Please open a [GitHub Issue](https://github.com/tdp1999/claude-code-ctx/issues) with:
- Plugin version (check `plugins/ctx/.claude-plugin/plugin.json`)
- Steps to reproduce
- Expected vs actual behavior
- Any error messages or logs

### Request a Feature

Open a [GitHub Issue](https://github.com/tdp1999/claude-code-ctx/issues) with the "enhancement" label describing your use case.

## What This Project Does NOT Support

- This is a Claude Code plugin, not a standalone application
- We do not provide support for Claude Code itself — see [Claude Code docs](https://docs.anthropic.com/en/docs/claude-code)
- For Anthropic API issues, contact [Anthropic Support](https://support.anthropic.com)
