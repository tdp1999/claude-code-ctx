# TDP Plugins - Claude Code Marketplace

> Curated Claude Code plugins for task-driven development and productivity

## 🚀 Quick Start

### Add the Marketplace

```bash
/plugin marketplace add tdp1999/tdp-plugins-marketplace
```

### Install Plugins

```bash
/plugin install ctx@tdp-plugins
```

## 📦 Available Plugins

### ctx - Task-Driven Development Workflow

Complete workflow management for task-driven development with `.context/` folder organization.

**Features:**
- 🎯 Project setup (vision, architecture, techstack)
- 📋 Planning tools (epic creation, investigation)
- 🔨 Task breakdown utilities
- 📊 Progress tracking with TDD support
- 🔄 Workflow automation

**Installation:**
```bash
/plugin install ctx@tdp-plugins
```

**Available Commands:**
- `/context-init` - Initialize new project structure
- `/vision` - Define project vision
- `/techstack` - Document technical stack
- `/architecture` - Design system architecture
- `/epic` - Create epic planning document
- `/breakdown` - Break epic into tasks
- `/task` - View/manage tasks
- `/start` - Begin working on task
- `/progress` - Track progress
- And more...
